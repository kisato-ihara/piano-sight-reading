# CLAUDE.md

## プロジェクト概要

ピアノの譜読み練習アプリ（個人利用）。
五線譜に表示された音符を見て、画面タップまたはMIDIキーボードで回答する音名当てクイズ形式。

## 技術スタック

- **フレームワーク**: React + Vite + TypeScript
- **楽譜描画**: VexFlow（五線譜・音符の描画）
- **音の再生**: Tone.js（Web Audio APIラッパー、ピアノ音源）
- **MIDI入力**: Web MIDI API（後から追加予定、Chrome Android対応）
- **データ保存**: IndexedDB（Dexie.js）- ローカル完結、サーバー不要
- **配信形態**: PWA（Progressive Web App）- ホーム画面に追加で利用
- **スタイリング**: 未定（TailwindCSS推奨）

## 対象デバイス

- Android スマートフォン（Chrome）
- MIDIキーボード: Casio CT-S1（USB OTG接続、後から対応予定）

## 機能要件

### フェーズ1（MVP）
- 五線譜に音符を1つ表示
- 画面上のピアノ鍵盤をタップして回答
- 正解/不正解のフィードバック + 正解音の再生
- モード選択:
  - 調性別（ハ長調モードなど）
  - 音域別（低音域だけモードなど）
- 回答時間・正誤の記録（IndexedDB）
- 苦手克服モード（ミスが多い音符を重点出題）

### フェーズ2（後から追加）
- MIDIキーボード入力対応（Web MIDI API）
- その他拡張（リズム、和音など）

## 開発方針

- 個人利用のため、配信・ストア公開は不要
- サーバー不要、ローカル完結
- バイブコーディングでサクッと作る
- コミットメッセージは日本語で記述
